# bcn-iot

### Install

1. Clone repo into `[DIR NAME]`
2. `$ cd [DIR NAME]`
3. `$ npm install` *you might need to run it twice, gulp-sass is currently buggy*
4. `$ node install.js [APP NAMESPACE]` *optional - just namespaces app in all coffee files*
5. `$ ./server/run`
* (new tab)
6. `$ gulp`
